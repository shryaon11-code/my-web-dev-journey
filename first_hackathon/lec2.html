HTML (HyperText Markup Language) is the foundation of every website. It gives structure to content, like arranging text, images, links, and media on a page. If the internet were a house, HTML would be the walls, rooms, and doors ‚Äî CSS would be the paint and decoration, and JavaScript would be the electricity.

Structure of an HTML Document

Every webpage follows a basic blueprint:

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>My First Page</title>
</head>
<body>
<h1>Hello FrogglePop!</h1>
<p>Welcome to the world of HTML.</p>
</body>
</html>

Key Parts:

  • <!DOCTYPE html> -- Declares this is HTML5.
  • <html> -- Root element of the page.
  • <head> -- Contains info like title, metadata, SEO.
  • <body> -- Contains everything the user sees.

Essential HTML Elements

Headings

  • <h1> to <h6> define headings.
  • <h1> is the main title; use only one per page.
  • <h1> -- <h6> are used for subtopics.

Example:

<h1>Main Title</h1>
<h2>Subtopic</h2>

Paragraphs

  • Written using <p> tag.
  • Browsers automatically add spacing around them.
<p>This is a paragraph of text in HTML.</p>

Links

  • Created with the (anchor) tag.
  • The href attribute defines the destination.
<a href="https://www.youtube.com/@CoderArmy9" target="_blank">Visit CoderArmy</a>

Images

  • Inserted with <img> tag.
  • Always include an alt attribute for accessibility.
<img src="frog.png" alt="Cute FrogglePop Frog" width="200">

Lists

Unordered List (<ul>) -- Bulleted

<ul>
<li>CSS</li>
<li>JavaScript</li>
<ul/>

Ordered List (<ol>) -- Numbered

<ol>
<li>Learn HTML</li>
<li>Learn CSS</li>
<li>Learn JS</li>
<ol/>

Other Useful Tags

  • Line Break: <br>
  • Horizontal Line: <hr>
  • Bold Text: <p> or <strong>
  • Italic Text: <i> or <em>
  • Comments (not visible on page):
<!-- This is a comment -->

Attributes in HTML

Attributes add extra details to tags.

Example:

<a href="page.html" target="_blank" title="Go to another page">Click Me</a>
  • href -- Link destination.
  • target="_blank" -- Opens in new tab.
  • title -- Tooltip text.

Best Practices for HTML

  1. Use semantic tags (<header>, <footer>, <article>) instead of only <div>.
  2. Always close your tags properly.
  3. Keep indentation clean (2‚Äì4 spaces).
  4. Use lowercase for tag names.
  5. Write descriptive alt text for images.
