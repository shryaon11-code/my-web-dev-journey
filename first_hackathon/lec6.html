Media Tag in HTML

<video> Tag

The <video> tag is used to embed videos.

Example:

<video src="intro.mp4" controls width="400"></video>
  • controls -- adds play, pause, volume buttons.
  • autoplay -- starts video automatically.
  • loop -- plays continuously.
  • muted -- starts without sound.
  • poster="image.jpg" -- adds a preview thumbnail before play.

Always provide multiple sources for browser compatibility:

<video controls>
<source src="intro.mp4" type="video/mp4">
<source src="intro.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

<audio> Tag

The <audio> tag embeds sound.

Example:

<audio controls>
<source src="song.mp3" type="audio/mpeg">
<source src="song.ogg" type="audio/ogg">
Your browser does not support the audio element.
</audio>
  • controls -- shows play/pause button.
  • loop -- repeat the sound.
  • autoplay -- play automatically (use carefully).

Pro Tip: Always provide fallback text ‚Äî some older browsers may not support these tags.

Multi-Page Website

A professional website is never just one page. Instead, it has:

  • Home
  • About
  • Contact
  • Service/ Lectures

These pages are connected through navigation links.

Example Navigation Bar:

<nav>
<a href="index.html">Home</a> |
<a href="about.html">About</a> |
<a href="lectures.html">Lectures</a> |
<a href="contact.html">Contact</a>
</nav>

Best practice: Save each page with .html extension and place all in one project folder.

Deploying Your First Project

Netlify

  1. Sign up at Netlify
  2. Drag and drop your project folder.
  3. Instantly get a free live link.

This method is free and biginner-friendly.

Pro Tips

  • Always compress videos before uploading (faster site load).
  • Store media in a separate /media or /assets folder.
  • Use relative paths (media/video.mp4) instead of absolute ones.
  • Add captions or transcripts for accessibility.
  • Start using good file naming: about.html, contact.html, style.css.

Mini Project / Practice Ideas

Practice is where you level up. Try these:

  • Build a 3-4 page personal portfolio:
    • Home -- Your intro
    • About -- Who you are
    • Projects -- Showcase work
    • Contact -- A form to connect
  • Add a video introduction on the home page.
  • Add background music or a podcast clip with <audio>.
  • Deploy the entire site on GitHub Pages or Netlify and share the link!

Homework

  1. Create a multi-page website with at least 3 pages.
  2. Add one video & one audio file.
  3. Deploy the project on GitHub Pages / Netlify.
  4. Share the live link with a friend for feedback.
